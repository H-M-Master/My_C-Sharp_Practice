## 20260123

WriteLine的使用，基本变量名称学习，方法类的创建与应用，常见关键字学习

### 1. 使用 VS Code 运行按钮（推荐）
- 打开 `Program.cs` 文件
- 点击右上角的三角形运行按钮 ▶️
- 程序会自动在终端中运行

### 2. 使用运行和调试
- 按 `F5` 或点击左侧调试图标
- 选择 ".NET Core Launch (console)"
- 程序会在调试模式下运行

### 3. 使用终端命令
```bash
dotnet run
```

## 项目结构

- `Program.cs` - 主程序入口
- `MyCSharpApp.csproj` - 项目配置文件
- `.vscode/` - VS Code 配置文件夹
  - `tasks.json` - 构建和运行任务配置
  - `launch.json` - 调试配置
  - `settings.json` - Code Runner 配置

## 修改代码

你可以直接在 `Program.cs` 中修改代码，然后使用上述任何方式运行。
